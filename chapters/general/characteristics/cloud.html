Cloud

Cloud

The framework is optimized to compile apps to AWS Lambda functions and spin up a serverless infrastructure via AWS Cloudformation. There are no servers to manage anymore, you only pay for what you use. Small Projects (~10,000 requests per day) are within AWS free tiers.
